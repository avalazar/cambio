# Cambio & Un-Solitaire Multiplayer Hub

A real-time, web-based multiplayer card game hub built to play Cambio and Un-Solitaire remotely with family. 

## Features
* **Real-Time Multiplayer:** Built with WebSockets for instant, low-latency game updates across different locations.
* **Two Distinct Game Modes:** * **Cambio:** A competitive memory and strategy game where players swap and peek at cards to end with the lowest score.
  * **Un-Solitaire:** A cooperative two-player variant of Solitaire focused on uncovering the entire deck.
* **Server-Authoritative State:** All game logic and deck management are handled securely on the backend to prevent client-side cheating or state desyncs.
* **Dynamic Lobbies:** Create private rooms with 4-letter join codes.

## Tech Stack
* **Backend:** Node.js, Express, Socket.io
* **Frontend:** Vanilla JS / HTML / CSS
